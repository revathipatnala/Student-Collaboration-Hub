# 🎓 Campus Network – Student Collaboration Platform

## 📌 Project Overview

Campus Network is a full-stack web platform designed for college students to access all campus services in one centralized system.

The platform allows students to report lost items, buy and sell products, join clubs, view announcements, and check upcoming campus events. It simplifies student communication and improves campus collaboration through a digital interface.

This project was developed using modern full-stack technologies as part of a Computer Science academic and practical learning project.

## ⚙️ Working of the Project

The Campus Network platform follows a client-server architecture.

• Users create accounts and login securely.
• Authentication validates user credentials stored in MongoDB.
• When users submit forms (lost item, product, event, etc.), data is sent to the backend API.
• The backend processes requests and stores information in the database.
• The frontend dynamically fetches stored data using API calls.
• All modules update automatically whenever new entries are added.

This creates a centralized digital ecosystem for student campus interaction.
